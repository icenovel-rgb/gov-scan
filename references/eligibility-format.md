# 자격요건.md — 회사 프로파일 양식

작업폴더에 두는 **회사 자격 프로파일**. 스킬 1단계에서 읽어 공고 신청자격과 대조한다.
자유서술도 허용하지만, 아래 키들이 채워져 있을수록 판정 정확도가 높다.

## 권장 양식

```markdown
# 자격요건 (회사 프로파일)

## 기본
- 상호:
- 대표자: (성명 / 연령 / 성별 — 청년·여성 우대사업 판정에 사용)
- 소재지: (시·도 / 시·군·구 — 지역제한 사업 판정에 핵심)
- 설립일: (YYYY-MM-DD)  → 업력 자동계산
- 사업자 형태: (개인 / 법인)

## 규모·업종
- 업종(업태/종목):
- KSIC(표준산업분류) 코드: (알면 기입)
- 연매출: (직전연도, 원)
- 상시 근로자수:
- 기업 유형: (예비창업 / 소상공인 / 중소기업 / 중견 / 벤처 / 이노비즈 / 메인비즈 / 사회적기업 / 1인기업 …)

## 인증·실적
- 보유 인증: (벤처/이노비즈/ISO/특허/기업부설연구소 …)
- 정부지원 수혜 이력: (중복수혜 제한 판정에 사용)

## 관심·우선순위
- 관심 분야: (R&D / 수출 / 자금 / 마케팅 / 시설 / 인력 / 창업 …)
- 제외 키워드: (관심 없는 분야 — 자동 스킵하지 않고 판정만 반영)
```

## 판정에서 자주 쓰는 기준

| 공고 제한 유형 | 프로파일에서 보는 값 |
|---|---|
| 지역 제한("○○시 소재 기업만") | 소재지 시·도/시군구 |
| 업력 제한("창업 7년 이내", "예비창업") | 설립일 → 업력 |
| 규모 제한("소상공인", "중소기업") | 기업 유형 / 근로자수 / 매출 |
| 업종 제한("제조업", "지식서비스") | 업종 / KSIC |
| 대표자 요건("청년", "여성", "39세 이하") | 대표자 연령·성별 |
| 인증 요건("벤처기업 대상") | 보유 인증 |
| 중복수혜 제한 | 정부지원 수혜 이력 |

## 판정 산출(스킬이 채움)
- `해당`: 명시 요건을 모두 충족(또는 제한 없음)으로 판단.
- `해당없음`: 하나라도 명백히 배제.
- `검토필요`: 공고 요약만으로 판단 불가 → 상세URL 확인 필요. DB엔 `보류`로 기록.
