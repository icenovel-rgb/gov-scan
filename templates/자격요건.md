# 자격요건 (창업자·기업 프로파일)

> gov-scan 스킬이 이 파일을 읽어 공고 신청자격을 자동 판정합니다.
> **신청주체를 여러 개 둘 수 있고, 그중 하나라도 자격이 맞으면 그 공고는 "해당"**으로 판정합니다.
> (예: 본인=기존 사업자 + 제자=예비창업자 → 둘 중 하나만 맞아도 해당)
> 아는 만큼만 채우면 되고, 비워두면 스킬이 발동 시 물어봅니다.

## 공통
- 소재지: (시·도 / 시·군·구)              ← 지역제한 사업 판정 핵심
- 업종(업태/종목):
- KSIC(표준산업분류) 코드:
- 관심 분야: (R&D / 자금 / 수출 / 마케팅 / 시설 / 인력 / 창업 …)
- 제외 키워드:                            ← 관심 없는 분야(판정엔 반영, 자동 스킵은 안 함)
- 정부지원 수혜 이력:                      ← 중복수혜 제한 판정용

- 사업자 형태: (개인 / 법인 / 없음)
- 설립일: (YYYY-MM-DD)                    ← 업력 자동계산 → 위 단계 자동매칭
- 대표자: (성명 / 연령 / 성별)             ← 청년(만39세↓)·여성 우대 판정
- 대표자 특이자격: (학생 / 교원 / 석·박사 / 경력단절 / 장애 …, 해당시)
- 기업유형·인증: (소상공인 / 중소 / 중견 / 벤처 / 이노비즈 / 메인비즈 / 소셜벤처 / 여성기업 / 1인창조기업 / 기술기업 …)
